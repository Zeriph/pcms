What is CSS?
CSS stands for Cascading Style Sheet and is a language that is primarily used in web sites to allow the site to define how it is presented (i.e. how it looks).


What does this mean exactly?
When you go to a web site you are actually downloading a file from a server that your web browser parses. Your browser will parse and display the information described in the file according the standards set forth by certain organizations; in the case of a web site, this is the HTML specification. HTML stands for Hyper Text Markup Language and defines how a web site presents information to the user as well as how information from the user is sent to the server, as in the case of a user logging in to a web site.

While the HTML specification does define certain visual elements, such as an img tag or the background property to change colors, it doesn't necessarily allow for an efficient way to style how a web site looks across the various pages. CSS can be included directly within the HTML or as a separate file that is downloaded, parsed and included where applicable by the web browser within a site.

A cascading style sheet allows a web developer to define how HTML elements (like a text box or a font) appear to the user, while separately defining the HTML layout within the document itself. The benefit of separating the visual styling from the layout, and centralizing it in one or more (.css) files, is that it allows a web developer to edit all visual or design elements within a site in one file (the .css file) without having to touch a lot of the HTML files themselves, thus allowing them to edit the way something looks without breaking how it works. The benefits to the user are that a single CSS file can be downloaded once by the web browser and be applied to any web page it is referenced in without having to download the CSS each time; in other words, CSS in a site allows faster page load times thanks to how browsers cache certain elements of a web site.
